📖 Cara Kerja SHA-256

1
Pre-processing

Input di-padding hingga panjangnya ≡ 448 (mod 512) bit

2
Inisialisasi

8 konstanta hash awal (H0–H7) dari bilangan prima

3
Kompresi

64 putaran operasi bitwise pada setiap blok 512-bit

4
Output

Gabungkan H0–H7 → 256 bit = 64 karakter hex

⚡ Avalanche Effect

Perubahan kecil pada input menghasilkan hash yang totally berbeda:

Hello → 185f8db32...
hello → 2cf24dba5...
↑ Hanya 1 huruf beda, tapi hash sama sekali berbeda!

✅ Properti SHA-256

✓
Deterministic

Input sama → hash selalu sama

✓
One-way

Tidak bisa di-reverse ke input asli

✓
Avalanche Effect

Perubahan kecil → output berubah drastis

✓
Collision Resistant

Sangat sulit menemukan 2 input dengan hash sama

⚠
Tanpa Salt

Tidak cocok untuk password tanpa salt tambahan
