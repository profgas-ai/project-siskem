Apa itu Hashing?

Hashing adalah proses mengubah data (plaintext) menjadi nilai tetap (hash/digest) menggunakan fungsi matematika satu arah. Hash tidak bisa di-reverse menjadi input aslinya.

One-way Deterministic Fixed-length
🔐

Bcrypt

Bcrypt dirancang khusus untuk hashing password. Secara otomatis menambahkan salt (nilai acak) ke setiap hash, sehingga password yang sama menghasilkan hash yang berbeda setiap kali.

  • ✓ Salt otomatis (anti rainbow table)
  • ✓ Cost factor yang bisa diatur
  • ✓ Lambat secara by-design (anti brute force)
  • ✓ Standar industri untuk password
🔷

SHA-256

SHA-256 (Secure Hash Algorithm 256-bit) adalah fungsi hash kriptografi dari keluarga SHA-2. Menghasilkan digest sepanjang 256 bit (64 karakter hex).

  • ✓ Output selalu 64 karakter hex
  • ✓ Avalanche effect (perubahan kecil = hash beda total)
  • ✓ Digunakan di blockchain, TLS, digital signature
  • ⚠ Tanpa salt, tidak ideal untuk password
⚖️
